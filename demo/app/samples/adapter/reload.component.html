The uiScroll augments the Datasource object passed from the outside during instantiating to provide an API to manipulate and assess the scroller. This API is accessible via special Adapter object. Currently it has only one method that allows you to reload the scroller: Adapter.reload(index). It resets the items buffer, resets the viewport params and starts fetching items

  • from index position (if reload is called with parameter)
  • from settings.startIndex position (if startIndex setting is set)
  • from the default position which is 1 (if neither setting nor parameter is present)
