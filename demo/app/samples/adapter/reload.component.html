The first Adapter API method is the reload method that allows us to reload the uiScroll: Adapter.reload(index). It resets the items buffer, resets the viewport params and starts fetching items

  • from index position (if reload had been called with index parameter);
  • from settings.startIndex position (if no index parameter is present and startIndex setting is set);
  • from the default position which is 1 (if neither setting nor parameter is present).
